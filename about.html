About CipherHack

CipherHack is an educational tool where learners can analyze how text is encrypted with a Vigenère cipher. Users can observe character frequencies and make assumptions in order to determine the encryption key within 10 guess.

Vigenère Cipher

The Vigenère cipher is a method of encrypting that uses multiple rotation ciphers where a repeating key is used to vary the rotation for each letter. This encryption was first created by Giovan Bellaso, but was misattributed to Blaise de Vigenère and had his name placed on it. See the example below for how it works.

Input
Message:  drink ovaltine
Key:      BENCH

Substitution
B (2):  d → e
E (5):  r → v
N (14): i → v
C (3):  n → p
H (8):  k → r
etc...

Output
Cipher: evvpr pznnajrr
